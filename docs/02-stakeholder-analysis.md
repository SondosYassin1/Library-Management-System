# 2. Stakeholder Analysis

## 2.1 Stakeholders Table

| Stakeholder | Role | Interest / Concern | Involvement Level |
|-------------|------|--------------------|-------------------|
| Admin | Manages the entire system including users, settings, and reports | Wants full control, accurate data, and system security | High |
| Library Staff | Handles daily operations such as adding books and managing borrowing/returning | Needs fast and easy system to reduce workload and errors | High |
| Member | Uses the system to search and borrow books | Wants easy access to books and clear information about availability | Medium |
| Guest | Browses available books without logging in | Wants simple and quick browsing experience | Low |
| Library Management | Oversees library performance and decision-making | Needs reports and statistics for planning and improvement | Medium |
